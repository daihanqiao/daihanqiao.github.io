前端Sublime Text 3插件推荐

  • 编辑器

  • Package Control ：包管理器

  • Emmet ：编码神器

  • ColorPicker ：颜色选择器

  • GitGutter ：git对比

  • All Autocomplete ：搜索所有打开的文件来寻找匹配的提示词

  • Terminal ：打开终端

  • SideBarEnhancements ：右键增强

  • BracketHighlighter ：在行号左侧显示当前所在的括号类型

  • TrailingSpaceS ：高亮空格

  • DocBlockr ：快速注释

  • AutoFileName ：自动提示路径

  • fileheader : 设置自定义模板及头部信息

  • SublimeTmpl : 自定义模板

  • SublimeCodeIntel ：代码提示

  • ClipBoard Manager ：粘贴板历史记录

  • Quote​HTML ：将html转成js字符串

  • CSScomb ：CSS属性排序

  • FileDiffs ：文件对比

  • Gutter Color ：css文件中左侧显示当前行所设置的颜色

  • IMESupoort ：中文输入输入框跟随

  • tortoiseSvn ：svn，右键文件夹或文件

  • SublimeLinter ：代码校验

  • Java​Script & Node​JS Snippets ：代码片段

  • autoprefixer ：自动补齐css前缀

  • Prettify ：格式化JS HTML CSS

  • Markdown Editing & Markdown Preview ：编辑预览markdown

  • Ctranslator tool：翻译工具

  • SideBarFolders ：管理项目文件夹

  • browser refresh ：刷新浏览器
